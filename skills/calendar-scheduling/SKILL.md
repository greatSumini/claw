# 캘린더 & 일정 협의 지침

## 일정 제안 포맷 (이메일/Discord 공통)
- 최소 2–3개 슬롯 제시 (유연성 확보)
- 형식: `YYYY-MM-DD (요일) HH:MM–HH:MM KST`
- 예: `2026-05-15 (금) 14:00–15:00 KST`

## 구글 캘린더 이벤트 생성 시
- 제목: `[미팅 유형] 참석자A × 참석자B` (예: `[Discovery Call] 김국진 대표 × Vooster`)
- 설명: 미팅 목적, 참석자 연락처, 사전 준비 사항
- 참석자 이메일이 있으면 초대장 발송 (`calendar.events.insert` → `sendUpdates: 'all'`)
- 화상회의 링크 자동 생성 여부 사용자에게 확인

## 이메일로 일정 잡을 때 흐름
1. 초안 이메일에 슬롯 2–3개 제시
2. 상대 확정 회신 후 → 캘린더 이벤트 생성 제안
3. 이벤트 생성 완료 후 → "통화/미팅 시간 확정 시 캘린더 일정도 바로 만들어드릴 수 있습니다" 안내 (아직 미확정이면)

## 시간대 주의사항
- 기본 KST (UTC+9)
- 해외 상대방이면 현지 시간대 병기
- 예: `14:00 KST (05:00 UTC)`
