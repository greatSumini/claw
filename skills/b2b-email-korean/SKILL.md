# 한국 B2B 이메일 초안 작성 지침

## 초안 작성 전 파악할 것
1. **수신자 정보**: 이름·직함·회사, 기존 거래처 여부
2. **톤**: 격식 / 캐주얼 / 중간 — context에서 추론 가능하면 먼저 제안하고, 불명확하면 한 번에 확인
3. **첨부 파일**: 있으면 제목줄에 [회사명] 태그 포함

## 말투 기준
- 캐주얼 요청 시: "~합니다" 어미 유지, "~고요," 허용
- "같아요" → "같습니다" (비즈니스 맥락 기본값)
- 문장 짧게, 단락 2~3줄 이하

## 서명 포맷 (기본)
감사합니다 :)
{이름} 드림
{소속/브랜드}

## 초안 전달 후 필수 안내
초안 출력 마지막에 반드시 추가: "수정사항 있으면 한 번에 정리해서 주시면 바로 반영하겠습니다"

## 문서/PDF 완성 직후 이메일 요청이 올 경우
- 발송 대상이 맥락에서 파악 가능하면 먼저 제안: "발송용 이메일 초안도 바로 드릴까요? (수신자: {추론된 이름})"
- 수신자 불명확하면 한 번에 확인 후 초안 작성
